# Web デザイン参照ガイド

Web デザイン（レイアウト・色・タイポグラフィ・モーション・UIの振る舞い）に関する判断は、
以下のガイドラインを参照すること。

## 基本参照（Web ファースト）

https://m3.material.io/foundations

- レイアウト
- 余白・グリッド
- カラーシステム
- タイポグラフィ
- モーション
- インタラクションの基本原則

Web UI の判断で迷った場合は、まずここを確認する。

## アクセシビリティ

https://m3.material.io/foundations/accessibility

- コントラスト
- フォーカス管理
- タッチターゲット
- 読みやすさ・操作しやすさ

アクセシビリティに関する判断は必ずこのページを参照する。

## タイポグラフィ

https://m3.material.io/foundations/typography

- 文字サイズ
- 行間
- 階層構造
- 可読性

文章量・情報密度に迷った場合はここを基準にする。

## 仕上がり品質・体験の確認（副参照）

https://developer.apple.com/design/human-interface-guidelines/foundations

- 情報の明瞭さ
- 視線誘導
- フィードバックの自然さ
- 全体の体験品質

実装後の「違和感チェック」「洗練度の確認」に使用する。
